Gold2iConomy by EdTheLoon

A Bukkit plugin that allows users to convert their gold bars (ingots) into money in two of the major currency systems.

Dependencies

Permissions (Not required) Either iConomy or BOSEconomy

Authors

EdTheLoon

Contact

EdTheLoon via Bukkit.org

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/EdTheLoon/Gold2iConomy
